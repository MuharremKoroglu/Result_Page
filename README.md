## <h2> About This App <img src = "https://c.tenor.com/JsoERRQcZqYAAAAi/thumbs-up-joypixels.gif" width = "42"></h2>
This app represents a result learning screen. Shows developers how to use the segue operation. First we created a new storyboard. Then we created a ViewController to which we will bind this interface. We started the segue process with the performSegue method. Then we checked the identifier of the screen to be segueed with using the segue.identifier method. In this way, a safer navigation was achieved. Then we cast the segue.destination method with the second ViewController and assign it to a variable. When we wrote the name of the variable and put a period, we were able to access the variables in the second viewController. In this way, we transferred user data. On the second page, we randomly showed the university logos on the screen using the arc4random method. Also, the username we got from the first page appears. Finally, we made some improvements and finished our application.
